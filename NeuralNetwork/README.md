# Binary Classification Neural Network from Scratch

## Objective

Implement a binary classification neural network using only NumPy.

## Features

- Weight Initialization
- Bias Initialization
- ReLU Activation
- Sigmoid Activation
- Forward Propagation
- Binary Cross Entropy Loss
- Backpropagation
- Gradient Descent
- Training Loop

## Comparison Models

- Logistic Regression
- Decision Tree
- Random Forest

## Dataset

Breast Cancer Wisconsin Dataset

## Results

The neural network was evaluated and compared with traditional machine learning models using Accuracy, Precision, Recall, and F1 Score.
